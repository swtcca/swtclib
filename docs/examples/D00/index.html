桌面应用说明 #

主要是指 electron, 重点在于把 web 应用打包成多种平台的桌面应用 #

  1. 进入 playground 目录
$ cd playground
  1. (可选)生成一个默认配置文件 package.json
$ npm init -y
  1. 安装@swtc/lib 库 和 electron 库
$ npm install @swtc/lib
$ npm install electron
